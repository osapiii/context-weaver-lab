# 設定と連携

AI や外部サービスの接続は、フッターまたはサイドバーから各設定画面を開いて管理します。

## よく使う設定

| 設定 | 用途 | 画面 |
| --- | --- | --- |
| AI 連携 | Gemini API キーなどのユーザー設定 | [設定](route:admin-preferences) |
| API キー | 外部 API を呼び出すためのキー管理 | [API キー](route:admin-api-keys) |
| 組織設定 | メンバー、Google 連携、送信元メール | [組織設定](route:admin-settings) |

## エラーが出るとき

- API キーが未設定または無効でないか確認します。
- Google Workspace 連携が切れていないか確認します。
- 組織メンバーの権限が足りない場合は管理者に確認してください。
