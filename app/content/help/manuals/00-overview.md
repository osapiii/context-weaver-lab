# EN AIstudio の全体像

EN AIstudio は、AI に仕事を依頼し、そのための知識を整え、実行ログを確認するための業務用ワークスペースです。

## 主な入口

| 入口 | 使う場面 | 開き方 |
| --- | --- | --- |
| 仕事をこなす | 経営相談、書類記入、調査、画像生成などを始める | [AI に依頼](route:admin-ai-studio) |
| AI を育てる | ナレッジ素材や取引先情報を登録する | [ナレッジ素材](route:admin-data-source) |
| 仕事ログ | AI の実行状況や完了した仕事を確認する | [仕事ログ](route:admin-workflow-executions) |
| 設定 | API キー、Google 連携、組織設定を管理する | [設定](route:admin-preferences) |

## 迷ったとき

右下の操作ガイドに「何をしたいか」をそのまま聞いてください。回答内に画面リンクが出た場合は、確認ボタンから該当画面を開けます。
