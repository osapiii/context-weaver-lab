SVG / Nanobanana refine 廃止

図解グローバルルール G1-G4

  • G1 平易な言葉: 横文字回避、専門用語は補足、数値は単位込み (label 16 字 / sub 30 字 / title 28 字以内)
  • G2 シンプルな見た目: 雲・有機形・キャラクター・絵文字は禁止。許可シェイプ (RECTANGLE / ROUNDED_RECTANGLE / OVAL / LINE / 矢印 / TRIANGLE / TRAPEZOID) のみ
  • G3 3 ステップ作成: 情報整理 → 型選択 → 配置と装飾 (色 3 色まで)
  • G4 70-25-5 色比: base 70% (canvas/gray) / brand 25% / accent 5% (アンバーは 1 挿絵に 1 箇所)

📦 シーン (50)

挿絵の「型」。NI.addNativeIllust(slide, sceneId, opts) で呼び出し。
